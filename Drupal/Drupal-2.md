# <center>Drupal Theme</center>

## 1.什么是Theme？

Theme 使 Drupal 网站以它自己的方式展现。每个 Theme 都是一组文件。Theme 通常是为 Drupal 站点定制代码的第一个地方之一，并且在多数情况下，它们是其创建的特定站点所独有的。

Drupal Theme 不是从头开始，而是从现有的HTML框架开始，并根据需要通过覆盖和更改必要的模板进行更改。

有些 Theme 只需修改几个选择位，而其他主题可能会选择覆盖几乎所有内容。无论哪种方式，只要它是HTML，就可以使用 Theme 更改它。

为了实现这一点，Drupal 需要在浏览器中显示内容的每个组件都为该元素提供了一个简单、最小的HTML模板。
